詳解

blast結果ファイルの参照側を変換してその値をカウントアップして出力

Example : A a -> taxid 123 A a -> taxid 123 A d -> taxid 234 A b -> taxid 123

  1. a,bをそれぞれ対応するidに変換する
  2. 同じid(a,a)は2度目に出てきた場合にカウントしない。
  3. a,bは変換後のidが同じであるためカウントアップ
  4. 変換後のidの数も出力する。

5. 出力は以下を参照

A 2 123:2,234:1
